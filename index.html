Опис предметного середовища

У цій лабораторній роботі розроблено вебзастосунок DragonCollect — онлайн-каталог для колекціонерів фігурок з мультфільму "Як приборкати дракона". Мета — допомогти колекціонерам візуально ідентифікувати свої фігурки, порівнювати схожі моделі та відмічати, які екземпляри вже є у колекції.

Інтерфейс дозволяє переглядати моделі за частинами мультфільму, додає можливість вибору "маю / не маю" для кожної фігурки, а також адаптується під розмір екрану.

HTML-код таблиці

<table>
  <tr><th>Назва</th><th>Частина</th><th>Стан</th></tr>
  <tr><td>Беззубик</td><td>1</td><td>Маю</td></tr>
</table>

Теорія: Таблиці створюються тегами <table>, <tr>, <td>, <th>. Атрибути: border, cellpadding, cellspacing, colspan, rowspan.

HTML-код форми

<form>
  <label>Пошук фігурки:</label>
  <input type="text" placeholder="Назва...">
  <button>Знайти</button>
</form>

Теорія: Форму задають тегом <form>, поля — <input>, кнопки — <button>, та <label> для підписів.

HTML-код зображення

<img src="img/toothless.png" alt="Беззубик" width="200">

Теорія: Зображення додається тегом <img>. Обов'язкові атрибути: src (шлях), alt (опис), опціональні: width, height, title.

СЕЛЕКТОРИ

Селектори тегу

CSS:

p {
                color: blue;
                font-weight: bold;
            }

HTML:

<p>Це абзац з синім жирним текстом</p>

Результат:

Це абзац з синім жирним текстом

Селектори класу

CSS:

.highlight {
                background-color: yellow;
            }

HTML:

<div class="highlight">Підсвічений елемент</div>

Результат:

Підсвічений елемент

Селектори ідентифікаторів

CSS:

                text-align: center;
