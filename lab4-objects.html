Завдання (варіант 5):

Напишіть функцію, яка приймає масив об'єктів і повертає новий масив. Зробіть знижку 20% на всі фрукти у масиві. Надайте id для кожного продукту.

Початковий масив:

const fruits = [
  { name: "apple", price: 200 },
  { name: "orange", price: 300 },
  { name: "grapes", price: 750 },
];

JavaScript-функція:

const fruits = [
  { name: "apple", price: 200 },
  { name: "orange", price: 300 },
  { name: "grapes", price: 750 },
];

const applyDiscountAndAddId = (items) => {
  return items.map((item, index) => ({
    id: index + 1,
    name: item.name,
    price: item.price * 0.8
  }));
};

const discountedFruits = applyDiscountAndAddId(fruits);
console.log(discountedFruits);
